# iWAF

Simple Web Application Firewall using Flask.

## Structure

- Web application firewall: `/iWAF`.
- Simple web server for demonstration: `/web_server`.

## Environment & Run

- Requirements:
    + docker
    + docker-compose


- Run iWAF server:

```
cd iWAF
docker-compose up
```

- Run web servers:

```
cd web_server
docker-compose up
```
